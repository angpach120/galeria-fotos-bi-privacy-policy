**Política de Privacidad**

Última actualización: 21/2/2026

Esta aplicación es una galería de imágenes utilizada únicamente con fines de visualización y análisis, integrada en Power BI.

Información que recopilamos

Esta aplicación no recopila, almacena ni procesa información personal de los usuarios.
No se solicitan ni registran datos como nombre, correo electrónico, ubicación, dirección IP ni ningún otro dato identificable.

Uso de imágenes

Las imágenes que se muestran en la aplicación se cargan mediante enlaces externos (URLs) proporcionados previamente.
La aplicación no aloja, descarga ni modifica dichas imágenes; únicamente las muestra como contenido visual.

Cookies y tecnologías de seguimiento

Esta aplicación no utiliza cookies, píxeles de seguimiento ni tecnologías similares.

Servicios de terceros

La aplicación se ejecuta dentro del entorno de Power BI.
El acceso y uso de Power BI están sujetos a las políticas de privacidad y términos de uso de Microsoft.
Los enlaces externos de imágenes dependen de los servidores donde estén alojados y se rigen por sus propias políticas.

Seguridad

Dado que la aplicación no recopila datos personales ni credenciales de usuario, no se almacena información sensible.

Cambios en esta política

Si en el futuro la aplicación incorpora funcionalidades que impliquen el manejo de datos personales, esta política será actualizada oportunamente.
